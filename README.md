# GPA Calculator

A clean GPA calculator built for students to calculate module GPA and overall GPA using cumulative credits and cumulative grade points.

## Features

- Module-based GPA calculator
- Overall GPA / CGPA calculator
- Responsive dark UI with Tailwind CSS

## Tech Stack

- React 18
- TypeScript
- Create React App (`react-scripts`)
- Tailwind CSS
- PostCSS + Autoprefixer

## Scripts

- `npm run dev` - start the development server
- `npm start` - start the development server
- `npm run build` - create a production build
- `npm test` - run the test suite

## Notes

The overall GPA formula used in this app is:

`CGPA = Total Cumulative Grade Points / Total Cumulative Credits`
